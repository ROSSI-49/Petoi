
简介
====

这是一个关于机器狗Petoi的小项目，为了利用实验室所购置的机器狗，拟利用此项目帮助同学们了解各种传感器和Arduino编程，openmv的使用，以及进行一些有趣的实验。本项目主要为科技节机器狗相关活动准备，但也整理了一些例程和相关的资料，供需要者进行学习。

本项目将会从以下的几个方面进行整理：

- Arduino环境准备
    + Arduino开发板介绍及相关资源
    + Arduino IDE安装与简介
- Petoi机器狗入门(施工中)
    + Petoi基本模块介绍
    + Petoi相关资料整理
    + 使用Petoi编程
- openmv入门(施工中)
    + openmv介绍
    + openmv代码示例

其中，我们主要参考了一下部分的资料：《Arduino程序设计基础（第二版）》，Petoi官方文档，Petoi论坛，Openmv官方资料，micor-python教程以及李荣仲老师在我院授课时的部分资料。篇幅所限，仅对其中内容做简单介绍，如要进一步了解相关内容，请通过上述方式查找学习。
